# AI 自动复盘 Badcase V0

## 1. Badcase 记录目标

记录 AI 自动复盘在真实或测试样本中的失败案例，用于推动 Prompt、输出结构、LLM 调用方式、错误兜底和数据聚合逻辑迭代。

## 2. 归因类型

| 归因类型 | 说明 |
| --- | --- |
| Prompt 约束不足 | Prompt 未明确限制输出或边界 |
| 输入数据不足 | 事件、标签、时长信息不足以支持结论 |
| 数据聚合错误 | 前端统计或聚合逻辑错误 |
| LLM 幻觉 | AI 编造不存在的信息 |
| 输出结构漂移 | AI 没有按约定结构输出 |
| 兜底缺失 | 异常场景没有合适提示 |
| 产品定义不清 | PRD 未明确当前场景应该如何处理 |

## 3. 严重程度

| 严重程度 | 说明 |
| --- | --- |
| P0 | 编造关键事实、误导用户，必须阻断上线 |
| P1 | 主要结论错误或结构不可用，必须优先修复 |
| P2 | 局部表达不准或建议泛化，需要迭代 |
| P3 | 轻微文案、格式或可读性问题 |
