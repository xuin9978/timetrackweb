# 聊天功能记录

这个文件夹用于记录聊天 Agent 功能从想法到实现过程中的变化、版本记录和产品思考。

## 当前文档

- `AI聊天Agent第一版PRD.md`：第一版个人时间 Agent 聊天功能 PRD。

## 建议记录内容

- 产品思考：为什么要做、解决什么用户问题、哪些能力暂时不做。
- 版本记录：每一版新增了什么、删减了什么、验证了什么。
- 设计决策：上下文、记忆、权限、聊天页面交互等关键取舍。
- 实现备注：接口、数据结构、模型调用和安全边界。

## 版本节奏

- V1：基于今天/本周日程和最近日记的上下文聊天。
- V1.5：聊天记录、上下文来源展示和右侧上下文面板。
- V2：长期记忆和用户可管理的个人偏好。
- V3：日程草案和确认后写入。
- V4：主动复盘、周期洞察和目标追踪。
